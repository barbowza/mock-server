# mock-server
A quick to configure and deploy mock API server to accelerate your Integrations

Proudly in PHP

## Features

- PHP 7.4 and above
- No dependencies/frameworks

# Testing

```shell
./vendor/bin/phpunit --testdox tests/unit
```

# Running

```shell
php -S 127.0.0.1:8765 -t /path/to/src/
```
